# holiday_dessert

## 參考網站 https://hackmd.io/@XchkZGnHT56qnXYLoFJM_A/H1KxBsilO

## 安裝vue-cli套件
```
npm install -g @vue/cli
```
## 解除安裝vue-cli套件
```
npm uninstall -g @vue/cli
```
## 建立vue專案 命令提示字元
```
vue create holiday_dessert
```
### 啟動專案前安裝套件
```
npm install
```
### 啟動專案
```
npm run serve
```
### 測試網址
```
http://localhost:8081/holidayDessert/index
```
### 發布專案 (建立在c:/xampp/htdcos裡面)
```
npm run build
```
### Lints and fixes files
```
npm run lint
```
